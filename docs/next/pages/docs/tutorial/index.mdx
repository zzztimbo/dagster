<p>If you're new to Dagster, we recommend working through this tutorial to become familiar with Dagster's feature set and tooling, using small examples that are intended to be illustrative of real data problems.</p>
<p>We'll build these examples around a simple but scary .csv dataset, <code>cereal.csv</code>, which contains nutritional facts about 80 breakfast cereals. You can find this dataset on <a href="https://raw.githubusercontent.com/dagster-io/dagster/master/examples/dagster_examples/intro_tutorial/cereal.csv">Github</a>. Or, if you've cloned the dagster git repository, you'll find this dataset at <code>dagster/examples/dagster_examples/intro_tutorial/cereal.csv</code>.</p>
